# AI Harness

A reusable, GitHub-native agent harness skeleton for turning messy software requests into clarified specs, vertical slices, isolated implementation loops, independent verification, and human-approved lessons.

Current skeleton version: `0.2.0`.

See `AIHARNESS-BUILD-PLAN.md` for the full vision and `docs/architecture.md` for the implemented skeleton layout.

## Apps

This repository also contains imported applications under `apps/`.

- `apps/dialectical-engine/`: local-first debate generation app for the
  simplified single-Mac `dezbatere.ro` setup. Start with
  `apps/dialectical-engine/README.md` and
  `apps/dialectical-engine/deploy/local-single-computer-dezbatere.md`.
