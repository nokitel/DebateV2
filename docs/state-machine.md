# State Machine

Status is canonical state. Labels are attributes only.

## Parent states

Draft → Researching → Clarifying → Critiquing → Plan Ready → Ready for Work → In Progress → In Review → Final Validation → Done.

Failure/escape states: Verification Failed, Rejected, Blocked.

## Sub-issue states

Queued → Implementing → Self-Verified → Verifying-Acceptance → Verifying-Quality → Verified → Integrating → In Review → Merged.

Failure/escape states: Fixing, Verification Failed, Rejected, Blocked.
